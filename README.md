# Rest Morty demo

## How to run this project

To run this project, follow these steps:

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the development server:

   ```bash
   npm run dev
   ```

## Used Libraries

- @tanstack/react-query
- localforage
- match-sorter
- react
- react-dom
- react-router
- react-router-dom
- sort-by
